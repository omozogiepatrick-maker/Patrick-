# MEO Maintenance Execution OS

A working MVP with assets, issue reporting, maintenance history, task management, notifications, reports, and an AI Decision Assistant.

## Deploy on Vercel

1. Get an API key at console.anthropic.com
2. Import this repo on vercel.com
3. Add ANTHROPIC_API_KEY in Project Settings, Environment Variables
4. Deploy

## Current limitations

Data storage is per browser, not shared across devices yet.
Login is not password protected yet.
